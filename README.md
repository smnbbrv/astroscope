# Astroscope

A collection of Astro integrations.

## Packages

| Package | Description |
|---------|-------------|
| [@astroscope/boot](./packages/boot) | Boot integration for Astro |

## Development

```bash
# Install dependencies
bun install

# Build all packages
bun run build

# Run demo app
bun run dev

# Run tests
bun test

# Typecheck
bun run typecheck
```
